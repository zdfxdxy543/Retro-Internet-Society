# 服务器运维工具示例：健康检查脚本

这是一个简单的Bash脚本，用于监控服务器基本状态，包括CPU、内存和磁盘使用情况。可以定期运行以维护服务器健康。

```bash
#!/bin/bash
echo '=== 服务器健康检查 ==='
echo '当前时间: ' $(date)
echo 'CPU使用率: ' $(top -bn1 | grep 'Cpu(s)' | awk '{print $2}')
echo '内存使用率: ' $(free -m | awk 'NR==2{printf "%.2f%%", $3*100/$2}')
echo '磁盘使用率: ' $(df -h / | awk 'NR==2{print $5}')
echo '检查完成。如有异常，请及时处理。'
```

将此脚本保存并设置为定时任务（如cron），可自动化运维监控。
